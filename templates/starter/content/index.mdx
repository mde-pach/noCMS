## One curated component library

Everything on this page is composed from `@nocms/components` — plain, typed Preact
components. The editor reads their prop types to generate controls automatically, so there's
no annotation language to learn and no JSON tree to hand-edit.

## Interactive islands

Most components render to static HTML for an instant first paint. The few that need
interactivity — declared with a single `island: true` flag — are hydrated in the browser
afterward, reusing the very same component. No second framework, no rerender of the whole
page.

## Theming is tokens, not code

Colors, fonts, spacing and radius live in a flat `theme.tokens` file and compile to CSS
variables at runtime. Editing a token restyles the whole site instantly — with no rebuild.
